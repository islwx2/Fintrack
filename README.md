# FinTrack 💰

Sistema de controle de finanças pessoais desenvolvido em **Java (console)**.  
O projeto permite registrar receitas e despesas e acompanhar automaticamente o saldo disponível.

## 📌 Funcionalidades

- Cadastro de **receitas**
- Cadastro de **despesas**
- **Cálculo automático do saldo**
- Listagem das movimentações financeiras
- Tratamento de erros com **exceções**

## 🛠 Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Listas (ArrayList)
- Tratamento de exceções

## 📂 Estrutura do Projeto

O sistema foi desenvolvido para execução em **modo console**, utilizando classes para organizar as regras de negócio e representar as movimentações financeiras.
